# 云南大学学位论文模板（YNU-thesis） - 非官方

- Author: 赵苡积
- Institution: 云南大学信息学院
- Version: V1.0

## 模板特色
- 支持的学位类型：博士、硕士、本科
- 支持的打印格式：单面、双面
> 单面格式：边距相同，内容居中，便于PDF阅读，适合论文撰写、匿名送审、在线送审时用此格式。
> 
> 双面格式：双面打印模式，在这种模式下奇偶页左侧边距不同，会自动生成空白页，确保新章节从奇数页码开始。




## 使用说明


- 编译引擎: XeLaTeX (推荐)
- 编译环境：在线可以用[overleaf](www.overleaf.com)；离线可以用TexLive+TexStudio
- type选项: Bachelor (学士), Master (硕士), Doctor (博士)
- print选项: twoside (双面打印), oneside (单面打印)

> 例如：\documentclass[type=Bachelor, print=twoside]{style/YNU-thesis}
